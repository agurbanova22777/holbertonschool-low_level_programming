# more_malloc_free

C project about dynamic memory allocation using `malloc` and `free` (Holberton/ALX style).

## Requirements
- Ubuntu 20.04 LTS
- `gcc` with: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Betty style (`betty-style.pl`, `betty-doc.pl`)
- No global variables
- No more than 5 functions per file
- Allowed C standard library functions: `malloc`, `free`
- You may use `_putchar` (provided by the checker)

## Files
- `0-malloc_checked.c` - `malloc_checked`: allocates memory; exits with status `98` if `malloc` fails.
- `main.h` - header file containing all function prototypes for this project.
