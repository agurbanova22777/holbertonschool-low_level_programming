# 0x0A. C - argc, argv

This project covers the concepts of command-line arguments in C.

## Requirements
- **Compiler:** `gcc` with flags `-Wall -Werror -Wextra -pedantic -std=gnu89`
- **Coding Style:** Betty style
- **Allowed Functions:** `_putchar`
- **Headers:** A `main.h` file with function prototypes is required.

### 0. What's my name?
A program that prints its name, followed by a new line.
